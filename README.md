# VestaCP PHP SDK

VestaCP SDK writted in PHP.

## Requirements:
- PHP 5.4 or earlier with:
  - cURL

## Installation:

```php
require_once 'src/vestaCPSDK.class.php';

$vestacp = new VestaCPSDK('hostname', 'username', 'password');
```
